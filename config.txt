# ============================================================
#  MISSÕES DE DESBLOQUEIO
#  Quando um item tem uma missão, o botão de download abre
#  um modal com tarefas. O link só é liberado após completar
#  todas as tarefas.
#
#  DOIS MODOS:
#
#  1. MISSÃO POR VERSÃO (cada botão tem tarefas diferentes):
#     [missao:NOME_DO_ITEM:NOME_DA_VERSAO]
#     Ex: [missao:bs_powerbrawl:V1]
#         [missao:bs_powerbrawl:V2]
#     O nome da versão deve ser IGUAL ao que está no campo versoes.
#     Ex: versoes = V1>>link | V2>>link   →  use V1 e V2 como chave
#
#  2. MISSÃO GERAL (todas as versões usam a mesma):
#     [missao:NOME_DO_ITEM]
#     Se uma versão não tiver missão própria, cai aqui.
#     Se não tiver nem a geral, o botão abre direto sem modal.
#
#  Campos da missão:
#  titulo / titulo_en       = título do modal
#  descricao / descricao_en = texto explicativo
#
#  Cada tarefa: [tarefa:NOME_DO_ITEM:VERSAO]  ou  [tarefa:NOME_DO_ITEM]
#  nome / nome_en   = nome da tarefa
#  dica / dica_en   = instrução pequena (opcional)
#  icone            = emoji
#  link             = URL que o botão abre
#  botao / botao_en = texto do botão
# ============================================================
